## 📁 Download

```bash

# Clone the repository
$ git clone https://github.com/KleytonLeite/next-tailwind-framermotion.git

# Enter into the directory
$ cd next-tailwind-framermotion

# Install dependencies
$ npm install

# Start the server
$ npm start

```
